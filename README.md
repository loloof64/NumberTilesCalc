# NumberTilesCalc

Reach the result with the given number tiles.

Maybe, if your solution starts well, it will also be able to give a solution based on your attempt.

## Artifacts

### Windows

- the zip file is an autonomous archive : just extract it and run the program
- the exe file is an installable version

### Linux

- the AppImage is autonomous archive : just give it execuatable rights and run it
- the DEB/RPM files are installable versions

## Developpers

### Translations

Generate translations with

```
dart run slang
```
